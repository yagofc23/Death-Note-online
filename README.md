# 📓 Death Note Online

Uma aplicação web interativa que recria a experiência do lendário caderno Death Note. Este projeto foi desenvolvido para demonstrar habilidades avançadas em front-end, oferecendo uma experiência imersiva através de áudio, animações e integrações reais.

## ✨ Funcionalidades

* **Manipulação de DOM:** Interatividade fluida e dinâmica com os elementos da página.
* **Áudio API:** Efeitos sonoros e trilha para recriar a atmosfera sombria do anime.
* **Animações CSS:** Transições e efeitos visuais detalhados na interface.
* **Integração com EmailJS:** Funcionalidade real para processar as "sentenças" escritas no caderno.
* **Design Responsivo:** Layout adaptado para funcionar perfeitamente em dispositivos móveis e desktops.
* **Easter Eggs:** Segredos escondidos na aplicação para os verdadeiros fãs da obra descobrirem!

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* [EmailJS](https://www.emailjs.com/)
